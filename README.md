# Daily Tasks Tracker

一個簡單的 Python CLI 待辦清單程式，用來練習 GitHub 操作。

## 功能
- 查看任務
- 新增任務
- 標記完成
- 刪除任務

## 安裝與使用
```bash
# 下載專案
git clone https://github.com/你的帳號/daily-tasks-tracker.git
cd daily-tasks-tracker

# 執行
python main.py
```

## 練習 GitHub 建議步驟
1. 建立新 branch
2. 加功能或修 bug
3. 提交 Pull Request
4. Merge 回 main
